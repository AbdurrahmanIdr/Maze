Maze Game

A thrilling 3D game built with raycasting

Map Exploration

Explore intricate maze-like levels filled with obstacles and hidden paths.

Enemy Encounters

Battle against challenging enemies who roam the maze and seek to impede your progress.

Player Abilities

Utilize unique abilities like sprinting and jumping to overcome obstacles and outsmart your opponents.

About the Project

The Maze Game is an exciting standalone project developed as part of the Holberton School curriculum. It showcases the implementation of raycasting technology to create a 3D gaming experience. The project was built with C programming language and utilizes the SDL2 library for graphics and user input handling.
